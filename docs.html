Apa itu Apollo AI?

Apollo AI Studio adalah platform AI pertama dari Indonesia yang dilengkapi fitur Global Chat — ruang ngobrol real-time antar pengguna. Dibangun dari nol oleh Tim Apollo AI (Iky, Lana, Deii), Apollo AI hadir dengan 14+ mode AI spesialis untuk berbagai kebutuhan: dari menulis skripsi, coding, sampai sekadar ngobrol santai.

Apollo AI tersedia gratis dan bisa diinstall sebagai aplikasi (PWA) langsung dari browser, tanpa perlu Play Store.

Mulai dalam 3 langkah

1 Buka apollo-ai-ochre.vercel.app di browser HP atau laptop kamu. Tidak perlu install dulu untuk mencoba.
2 Daftar akun gratis dengan email, atau langsung masuk pakai Google. Akun diperlukan untuk menyimpan riwayat chat dan akses Global Chat.
3 Pilih mode AI yang sesuai kebutuhanmu (atau biarkan default), lalu mulai ngobrol. Bisa kirim teks, foto, atau file PDF.

Mode AI yang tersedia

Apollo AI punya 14+ mode spesialis, antara lain: Mode Skripsi untuk bantuan akademik, Mode Coding untuk menulis dan debug kode, Mode Ujian untuk latihan soal, Mode Kreatif untuk cerita dan konten, Mode Santai untuk ngobrol bebas, dan beberapa mode lain seperti hukum dan bisnis. Setiap mode menyesuaikan gaya jawaban Apollo AI agar lebih relevan dengan topik yang kamu bahas.

Fitur lainnya

Selain chat satu-ke-satu dengan AI, Apollo AI juga punya Global Chat — ruang ngobrol real-time dengan semua pengguna lain, PDF Analyzer untuk upload dan tanya jawab soal isi dokumen, Vision AI untuk menganalisis foto yang kamu kirim, serta Live Chat untuk menghubungi admin secara langsung jika ada masalah.
